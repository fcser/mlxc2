# 接口文档

## 一.商品抢单模块

### 1.商家操作

#### 1.1插入商品信息

- 接口(json)

  ```
  /addProduct.do
  ```

- 请求参数

  | 参数名称        | 参数类型    | 是否必选 | 参数说明 |
  | ----------- | ------- | ---- | ---- |
  | name        | String  | 是    | 产品名称 |
  | description | String  | 否    | 产品说明 |
  | stock       | Integer | 是    | 库存量  |
  | price       | Double  | 是    | 定价   |
  | discount    | Integer | 否    | 折扣   |

- 返回参数

  ```json
  {
      "code":200,
    	"msg":"success"
  }
  ```

#### 1.2修改商品信息

- 接口（json)

  ```
  /updateProduct.do
  ```

- 请求参数

  | 参数名称        | 参数类型    | 是否必选 | 参数说明 |
  | ----------- | ------- | ---- | ---- |
  | id          | Long    | 是    | 商品id |
  | description | String  | 否    | 产品说明 |
  | stock       | Integer | 否    | 库存量  |
  | price       | Double  | 否    | 定价   |
  | discount    | Integer | 否    | 折扣   |

- 返回参数

  ```json
  {
      "code":200,
    	"msg":"success"
  }
  ```

#### 1.3删除商品

  ```
  /delProduct.do
  ```

- 请求参数

  | 参数名称 | 参数类型 | 是否必选 | 参数说明 |
  | ---- | ---- | ---- | ---- |
  | id   | Long | 是    | 商品id |

  ​

- 返回参数

  ```json

    	"msg":"success"
  ```

#### 1.4商品列表

- 接口

  ```
  /listProduct.do
  ```

- 请求参数

  | 请求参数     | 参数类型    | 是否必选 | 参数说明       |
  | -------- | ------- | ---- | ---------- |
  | name     | String  | 否    | 商品名称（模糊查找） |
  | pageNum  | Integer | 否    |            |
  | pageSize | Integer | 否    |            |

- 返回参数

  ```json
  {
      "code":200,
    	"msg":"success",
    	"data":[
          {
              "id":1,
            	"name":"蓝莓",
            	"decription":"好吃的",
            	"stock":100,//目前库存
            	"price":80,
            	"discount":7,//折扣
            	"num":999 //订单量
          },{
              "id":1,
            	"name":"蓝莓",
            	"decription":"好吃的",
            	"stock":100,//目前库存
            	"price":80,
            	"discount":7,//折扣
            	"num":999 //订单量
          }
      ]
  }
  ```

#### 1.5订单列表

- 接口

  ```
  /listRecords.do
  ```

- 请求参数

  | 请求参数      | 参数类型    | 是否必选  | 参数说明         |
  | --------- | ------- | ----- | ------------ |
  | id        | Long    | 是     | 商品id         |
  | userPhone | String  | 否     | 用户手机号码（模糊查找） |
  | pageNum   | Integer | false |              |
  | pageSize  | Integer | false |              |

- 返回参数

  ```json
  {
    "code":200,
    "msg":"success"
    "data":[{
      "userName":"hehe",
      "userPhone":"15757575757",
      "productName":"蓝莓",
      "count":5,  //订购数量
      "decription":"sadf", //订单备注
    	"sum":400   //总价
    }]
  }
  ```
